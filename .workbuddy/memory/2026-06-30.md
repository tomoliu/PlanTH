## 修复：角色选择背景图不切换
- 问题：角色选择场景进入/切换时背景图未按 JSON 中的 `lobby_background` 切换。
- 使用 Godot 4.7 无头模式进行烟雾测试，发现根因：`JSON.parse_string` 返回的 `Dictionary` 元素在被 `Array[Dictionary]` 接收时，因类型转换时序导致 `_characters` 数组偶发为空。
- 修改 `Scenes/CharacterSelect/character_select.gd`：将循环内 `character as Dictionary` 改为 `var character: Dictionary = raw_character`，并把 `Dictionary` 元素直接追加到 `_characters`；同时为 `_select_character` 添加越界检查，缺失 `lobby_background` 时给出警告。
- 验证：无头模式脚本加载后，默认显示 Reimu 背景；调用 `_select_character(1)` 后成功切换到 Marisa 背景，无报错。
